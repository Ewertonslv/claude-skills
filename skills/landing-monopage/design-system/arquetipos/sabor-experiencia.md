# Arquétipo — Sabor & Experiência (alimentação e eventos)

**Cena:** alguém escolhendo onde comer/comemorar pelo apetite e pela vibe — decide em segundos olhando foto. Compra desejo sensorial + ocasião.

**Lane estética:** image-led (a foto é o design), cor com apetite.
- **Anti-clichê 1ª ordem:** vermelho-fome + foto stock de prato genérico; script "handmade" pra tudo.
- **Anti-clichê 2ª ordem:** o rústico-toscana-bege (madeira + oliva + serifada fina) que virou default de "restaurante artesanal". Se o lugar não é literalmente rústico, não vista o figurino.

## Combos

### 1. `brasa` — dark quente + ember (default p/ restaurante)
```css
--bg:#171210; --surface:#221a16; --ink:#f6ece2; --muted:#c2ab97;
--brand:#d4703a; --accent:#8a9b62; --brand-rgb:212,112,58;
```
Fontes: **Yeseva One** (display) + **Asap** (corpo)
`https://fonts.googleapis.com/css2?family=Yeseva+One&family=Asap:wght@400;500;600&display=swap`
Quando usar: jantar/carnes/massas; noite, brasa, aconchego adulto.

## Ênfase de seções + prova
**Herói full-bleed com foto do prato/evento** (a foto É o herói; overlay de texto) · Cardápio/menu resumido ou "especialidades" (link p/ cardápio completo/iFood quando houver) · **Prova = galeria de pratos/eventos é o pico** · Depoimentos/avaliações (Google) · Contato com **endereço + horário + mapa em destaque** (negócio de localização!) · reserva/pedido via WhatsApp.

## Device de assinatura
Foto sangrada (full-bleed) com tipografia display sobreposta; divisores finos estilo menu.

## Tom de copy + CTA
Sensorial e direto ("massa fresca todo dia"), sem adjetivo vazio. CTA: **"Reserve pelo WhatsApp"** / **"Peça agora"**.

## Imagem
Obrigatória e decisiva: comida/eventos reais, luz natural/quente. Zero banco de imagem — foto ruim aqui mata a venda; sem foto boa, adie o launch e peça fotos.
