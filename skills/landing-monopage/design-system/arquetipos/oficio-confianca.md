# Arquétipo — Ofício & Confiança (serviços técnicos e criativos)

**Cena:** alguém com um problema prático (carro quebrado, obra, pet, registro de um momento) procurando alguém que **resolve bem e não enrola**. Compra competência demonstrável + preço justo.

**Lane estética:** honesto-competente; mostra o trabalho, não o discurso.
- **Anti-clichê 1ª ordem:** ferramenta cruzada + amarelo-alerta (ofícios); marca d'água + câmera (fotógrafo); planta baixa de stock (arquiteto).
- **Anti-clichê 2ª ordem:** o "industrial-dark-bold" uniforme pra todo serviço técnico. Mecânica ≠ pet shop ≠ ateliê — a família compartilha honestidade, não figurino.

## Combos

### 1. `oficina` — grafite + laranja segurança (default p/ mecânica, serviços residenciais)
```css
--bg:#141619; --surface:#1d2126; --ink:#eef1f4; --muted:#9aa3ad;
--brand:#f2762e; --accent:#5e88a8; --brand-rgb:242,118,46;
```
Fontes: **Barlow Condensed** (display) + **Barlow** (corpo)
`https://fonts.googleapis.com/css2?family=Barlow+Condensed:wght@600;700&family=Barlow:wght@400;500;600&display=swap`
Quando usar: oficina, elétrica, hidráulica, manutenção; competência industrial.

### 2. `quintal` — claro + teal/coral amigável (default p/ pet shop, banho e tosa)
```css
--bg:#faf6ec; --surface:#ffffff; --ink:#2b2b23; --muted:#5f5f52;
--brand:#2f8f83; --accent:#e0703c; --brand-rgb:47,143,131;
```
Fontes: **Baloo 2** (display, arredondada) + **Nunito Sans** (corpo)
`https://fonts.googleapis.com/css2?family=Baloo+2:wght@600;700&family=Nunito+Sans:wght@400;600;700&display=swap`
Quando usar: pets, serviços de família; caloroso sem ser infantil.

### 3. `atelie` — galeria quase-mono (default p/ fotógrafo, arquiteto/interiores)
```css
--bg:#f7f6f3; --surface:#ffffff; --ink:#191918; --muted:#55554f;
--brand:#1f1f1d; --accent:#c05a3a; --brand-rgb:31,31,29;
```
Fontes: **Hanken Grotesk** — família única, contraste por peso (700-800 display / 400 corpo)
`https://fonts.googleapis.com/css2?family=Hanken+Grotesk:wght@400;500;700;800&display=swap`
Quando usar: portfólio visual — o site some e o trabalho aparece; acento usado raríssimo.

## Ênfase de seções + prova
Herói direto (o que faz + para quem + CTA) · **Prova = o pico**: galeria de trabalhos (fotógrafo/arquiteto/tosa) ou antes/depois de serviço (funilaria, reforma) · Serviços em lista editorial com escopo claro ("o que está incluso") · Depoimentos/avaliações Google · Contato com **área de atendimento** (bairros/cidades) + horário.

## Device de assinatura
`oficina`: numeração real de processo (diagnóstico → orçamento → execução). `atelie`: grid de galeria assimétrico. `quintal`: cantos arredondados generosos.

## Tom de copy + CTA
Claro, sem jargão, orientado a resultado e prazo ("orçamento sem compromisso", "prazo combinado é prazo cumprido"). CTA: **"Peça um orçamento"** / fotógrafo: **"Consulte datas"**.

## Imagem
Trabalho real (oficina/obra/ensaio/pets atendidos). Para `atelie` a imagem é o produto — mesma regra do sabor-experiencia: sem foto boa, não lança.
