# Arquétipo — Desejo Visual (beleza e estética pessoal)

**Cena:** alguém querendo se sentir mais bonito(a) e confiante, escolhendo pelo olho: o visual do site É a amostra do serviço. Compra desejo + identificação de estilo.

**Lane estética:** committed/drenched com personalidade ("dopamine design" controlado).
- **Anti-clichê 1ª ordem:** rosa-bebê + script cursivo + florzinha (salão); barber pole + madeira rústica + navalha (barbearia).
- **Anti-clichê 2ª ordem:** dark-luxo-dourado genérico igual pra tudo. A cor tem que vir do estilo do profissional, não do "premium default".

## Combos

### 1. `vinho-terracota` — quente profundo (default p/ salão, maquiadora)
```css
--bg:#1a1013; --surface:#241519; --ink:#f7ece9; --muted:#c9a99f;
--brand:#c8695a; --accent:#e6b855; --brand-rgb:200,105,90;
```
Fontes: **Bricolage Grotesque** (display) + **Figtree** (corpo)
`https://fonts.googleapis.com/css2?family=Bricolage+Grotesque:opsz,wght@12..96,600;12..96,700&family=Figtree:wght@400;500;600&display=swap`
Quando usar: beleza feminina adulta, vibe caloroso-sofisticada.

### 2. `neon-noite` — near-black + elétrico (default p/ barbearia, tatuagem)
```css
--bg:#0d0d10; --surface:#16161b; --ink:#f2f2f4; --muted:#a3a3ad;
--brand:#ef5a78; --accent:#46e0b0; --brand-rgb:239,90,120;
```
Fontes: **Unbounded** (display) + **Sora** (corpo)
`https://fonts.googleapis.com/css2?family=Unbounded:wght@500;600;700&family=Sora:wght@400;500;600&display=swap`
Quando usar: público jovem/urbano, atitude; estúdio com identidade forte.
(Barbearia mais clássica: troque `--brand` p/ um âmbar #d99a4e e mantenha o resto.)

### 3. `pastel-luxo` — claro lavanda/champagne (default p/ lash/nail designer)
```css
--bg:#f7f2f4; --surface:#ffffff; --ink:#2a2430; --muted:#665d6b;
--brand:#8e5aa8; --accent:#c99a5b; --brand-rgb:142,90,168;
```
Fontes: **Prata** (display) + **Jost** (corpo)
`https://fonts.googleapis.com/css2?family=Prata&family=Jost:wght@400;500;600&display=swap`
Quando usar: delicadeza calma-luxo (cílios, unhas, sobrancelha).

## Ênfase de seções + prova
Herói com **foto do trabalho/espaço** (não só retrato) · Serviços **com preço/duração quando o cliente topa** · **Prova = galeria/portfólio é o pico** (grid de trabalhos reais; antes/depois quando couber) · Depoimentos com nome/foto · Contato + "como chegar".

## Device de assinatura
Um bloco/drench de cor forte numa seção; hover vivo nos itens da galeria.

## Tom de copy + CTA
Caloroso, confiante, aspiracional sem ser brega; 2ª pessoa. CTA: **"Agende seu horário"**.

## Imagem
Essencial — sem foto real de trabalho o nicho não converte. Nunca substituir galeria por blocos de cor. Fotos autorizadas de clientes.
